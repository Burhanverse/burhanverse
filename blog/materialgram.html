Setup For Arch users:

NOTE: If you have installed the flatpak version then please do remove it first.
  • ⦿ You can use this installer script:
      • sudo sh -c "curl -s https://raw.githubusercontent.com/materialgram/arch/x86_64/installer.sh | bash"

Setup RPM Repository:

NOTE:

⦿ If you have installed the flatpak version then please do remove it first.

⦿ The github repo is now archived and is replaced with the materialgram-rpm which provides the spec file for the builds on fedora copr project. The entire process is now automated and packages are OpenPGP/GPG signed by fedora with added supports for X86_64 versions of Centos-stream 10, Centos-stream 9, Fedora 40, Fedora 41, Fedora rawhide, openSUSE Leap 15.5, openSUSE Tumbleweed, RHEL 8 & RHEL 9.

  • ⦿ Open a new terminal and copy-paste the below content to add the RPM repo of Materialgram:
      • sudo dnf copr enable burhanverse/materialgram
    • ⦿ Old users must remove the materialgram.repo file from thier yum.repo.d directory before using the new materialgram copr project repo:
      • sudo rm -rf /etc/yum.repos.d/materialgram.repo
    • ⦿ Then to install simply run:
      • sudo dnf install materialgram
    • ⦿ To update simply type:
      • sudo dnf update materialgram --refresh
    • ⦿ To uninstall the rpm Package:
      • sudo dnf remove materialgram
    • ⦿ Old Installer(based on github repo) just for reference:
      • sudo bash -c "$(curl -fsSL https://raw.githubusercontent.com/burhancodes/materialgram-repo/main/install.sh)"

Setup DEB Repository:

Note: If you have installed the flatpak version then please do remove it first.

  • ⦿ Open a new terminal and copy-paste the below content to add the RPM repo of Materialgram:
      • bash <(curl -s "https://raw.githubusercontent.com/ghazzor/materialgram-deb-package/master/install.sh")
      OR wget -qO- https://raw.githubusercontent.com/ghazzor/materialgram-deb-package/master/materialgram_repo.asc | sudo tee /etc/apt/trusted.gpg.d/materialgram_repo.asc echo "deb [arch=amd64] https://raw.githubusercontent.com/ghazzor/materialgram-deb-package/master/apt/repo/ bionic main" | sudo tee /etc/apt/sources.list.d/materialgram.list
  • ⦿ Then to install simply run:
      • sudo apt update && sudo apt install materialgram
