RSS-ify

  • RSS-ify is a lightweight telegram bot written in Javascript and powered by NodeJS™, that helps you stay updated by delivering content from your favorite feeds directly to your group chat or in private. Customize, organize, and manage your subscriptions effortlessly!
  • RSS-ify is available @rssifyxbot

    • Last updated: 9th Dec, 2024

Features:

  • ⦿ RSS-ify supports group topics, which means you can easily set a particular topic in your group as default for your feed updates just by using the /set command.
  • ⦿ Atom, RSS 2.0 and RSS 1.0 are supported.
  • ⦿ The bot checks for new updates for your feeds every 60 seconds.
  • ⦿ If your site is protected with cloudflare then the bot will return invalid URL or an error.
  • ⦿ As of now channels are not supported, but I do have a plan to add support in the future.
    • NOTE: The source code is currently private, but I have plans to make it open-source in the future.

Available commands:

  • ⦿ /set command is used to set a topic for the feed updates ( only available for groups with topics enabled )
  • ⦿ /add to add new feeds.
  • ⦿ /del to delete a feed.
  • ⦿ /list command shows the list of subscribed feeds
  • ⦿ /ping to check bot server ping.
  • ⦿ /about shows the current bot version, description, author, etc.
